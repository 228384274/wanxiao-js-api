# wanxiao-js-api
-玩校JSSDK  v1.1.8, v1.1.9版本
-原理 web页面用js方法通过 jsbrige与原生系统进行通信,定义了一系列的原生功能列表,满足web页面使用需求.
#
-v1.1.8 JSSDK版本支持目前玩校v3.2.7及以上版本
-v1.1.9 JSSDK版本支持目前玩校v3.3.0及以上版本
